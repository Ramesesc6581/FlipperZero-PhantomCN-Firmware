<h2>🎮 How to Use the Extra Tools</h2>
<h3>BLE Killer</h3>
<p>Open the <strong>Apps</strong> menu. Scroll to <strong>Bluetooth</strong> and select <strong>BLE Killer</strong>. Point your Flipper at a BLE device (like a smart watch or speaker) and press the OK button. The tool sends packets that may cause the device to disconnect or crash. Use it only on devices you own.</p>

<h3>Sub‑GHz PT2262 Brute Force</h3>
<p>Go to <strong>Sub‑GHz</strong> &gt; <strong>Brute Force</strong>. Select <strong>PT2262</strong>. The device will try different addresses and keys. When it finds a match, it beeps. This is useful for testing your own garage door or gate remotes.</p>

<h3>Signal Packs (Tesla, RF Jam, etc.)</h3>
<p>These are pre‑loaded when you install the firmware. Open <strong>Sub‑GHz</strong> &gt; <strong>Saved Signals</strong>. You will see folders like <strong>Tesla</strong>, <strong>RF Jam</strong>, and <strong>Misc</strong>. Pick one and press OK to transmit the signal.</p>

<h2>🔧 Troubleshooting</h2>
<ul>
  <li><strong>My Flipper does not show the FLIPPER drive.</strong> Try a different USB cable. Make sure the cable can transfer data, not just power.</li>
  <li><strong>The Chinese text looks like boxes or gibberish.</strong> Make sure you copied the <code>firmware.bin</code> file correctly. If the problem continues, download the firmware again and reinstall.</li>
  <li><strong>The update fails with an error.</strong> Your Flipper battery might be low. Charge it above 50% and try again.</li>
  <li><strong>I cannot find the update option on the Flipper.</strong> Press the Left button to go back to the main menu. Look for <strong>系统更新</strong> (System Update) or <strong>更新</strong> (Update).</li>
  <li><strong>Stuck after update – screen is black.</strong> Hold the <strong>Back</strong> button and press the <strong>Reset</strong> button on the back of the Flipper. The device should reboot.</li>
</ul>
